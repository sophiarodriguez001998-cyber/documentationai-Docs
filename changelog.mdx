## Version History

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>
  ## New Features

  - Added interactive tree view for navigating deeply nested JSON structures.
  - Introduced real-time syntax highlighting with 10+ color themes.

  ## Improvements

  - Enhanced validation engine to detect schema mismatches `{required: true}` faster.
  - Performance boost: Large JSON files (`>1MB`) now load `<2s`.

  ## Bug Fixes

  - Fixed collapse/expand states not persisting in tree view.
  - Resolved copy-to-clipboard issues on mobile devices.
</Update>

<Update label="2024-09-20" description="v1.2.0" tags={["feature", "bugfix"]}>
  ## New Features

  - Schema validation integration: Paste a JSON schema and validate against it.
  - Export options: Download beautified JSON as formatted file or shareable link.

  ## Improvements

  - Beautify now supports custom indentation (2-8 spaces).
  - Search functionality highlights all matches across the entire document.

  ## Bug Fixes

  - Corrected parsing errors for JSON with trailing commas.
  - Fixed infinite loading spinner for malformed inputs.
</Update>

<Update label="2024-08-10" description="v1.1.0" tags={["improvement", "bugfix"]}>
  ## New Features

  - Dark mode toggle for better low-light viewing.
  - Keyboard shortcuts: `Ctrl+B` to beautify, `Ctrl+V` to validate.

  ## Improvements

  - UI redesign for faster navigation between tabs (View, Validate, Beautify).
  - Added line numbers and folding for code-like JSON editing.

  ## Bug Fixes

  - Patched security vulnerability in URL import handling.
  - Stabilized drag-and-drop file upload for all browsers.
</Update>
