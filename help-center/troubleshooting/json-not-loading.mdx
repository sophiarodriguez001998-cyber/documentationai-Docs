---
title: JSON Not Loading Properly
description: Fix issues when JSON fails to display or load.
---

## Symptoms

- Blank viewer
- Error: "Failed to parse JSON"

## Common Causes

- Invalid JSON format
- File too large (over 5MB)

## Quick Fix

<Steps>
  <Step title="Check Your JSON">
    ````json
    Ensure your input matches correct JSON syntax. Use curly braces { } and double quotes.
    ````
  </Step>
  <Step title="Reduce File Size">
    Try splitting large JSON files or use a smaller sample for upload.
  </Step>
</Steps>
