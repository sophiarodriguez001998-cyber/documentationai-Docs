## Overview

Validating JSON checks for formatting errors and highlights problems in your data.

## Steps

1. Paste your JSON into the input box on the homepage.
2. Click the **Validate** button below the editor.
3. Errors will appear if the JSON is invalid; otherwise, you'll see a "Valid JSON" confirmation.

## Example

```json
{
  "name": "Alice",
  "age": 25
}
```
