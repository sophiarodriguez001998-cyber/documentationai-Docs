## Overview

Use Json Viewer AI to format and explore JSON with ease.

## Steps

<Steps>
  <Step title="Paste or Upload JSON">
    ````json
    Drag a file or paste JSON into the main text area.
    ````
  </Step>
  <Step title="Click Beautify">
    Press the **Beautify** button below the editor to format and indent the JSON content.
  </Step>
  <Step title="Explore in Tree View">
    Switch to **Tree View** for easy navigation through nested data.
  </Step>
</Steps>
